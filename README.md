# AI Assistant - Modern Chat Interface

## 🤖 Overview

A sophisticated AI chat application featuring a clean, modern design inspired by leading AI platforms. Built with vanilla JavaScript and styled for optimal user experience across all devices.

## ✨ Features

### 🎨 Modern Interface
- **Clean, minimalist design** following industry best practices
- **Responsive layout** that works seamlessly on desktop, tablet, and mobile
- **Professional color scheme** with high contrast for readability
- **Smooth animations** and micro-interactions
- **Custom scrollbars** matching the design theme

### 🧠 AI Model Selection
Choose from 7 different AI models:
- **Qwen-Local** - Fast and efficient
- **KIMI-Local** - Advanced reasoning
- **Qwen3-14B-Local** - High-capacity model
- **GLM-4.7-Local** - Balanced performance
- **step-3.5-flash-Local** - Quick responses
- **GLM-4.7-Flash-REAP** - Optimized for speed
- **ggml-org-gpt-oss-120b** - Open source powerhouse

### 💬 Chat Features
- **Real-time messaging** with instant AI responses
- **Conversation history** maintained during session
- **Typing indicators** showing AI is processing
- **Message bubbles** with distinct user/AI styling
- **System messages** for model changes and status updates
- **Error handling** with user-friendly feedback

### 🎯 User Experience
- **Keyboard shortcuts** - Enter to send, Shift+Enter for new line
- **Auto-focus** on input field for seamless chatting
- **New chat button** to start fresh conversations
- **Loading states** and visual feedback
- **Accessibility features** with semantic HTML

## 🛠️ Technical Architecture

### 🏗️ Frontend Stack
- **Pure HTML5** with semantic markup
- **Modern CSS3** with flexbox and grid layouts
- **Vanilla JavaScript** - No framework dependencies
- **Progressive enhancement** - Works with JavaScript disabled
- **Cross-browser compatibility** - Chrome, Firefox, Safari, Edge

### 🔧 Configuration System
- **Modular config.js** for AI model endpoints
- **Obfuscated configuration** for security
- **Dynamic model switching** without page reload
- **Fallback handling** for connection issues

### 🎨 Design Principles
- **Mobile-first responsive design**
- **WCAG accessibility compliance**
- **Performance optimized** with minimal animations
- **SEO friendly** with proper meta tags
- **Security hardened** with content protection

- **Secondary**: #F3F4F6 (Light gray)

- **Headings**: 600 weight for hierarchy
- **Body**: 400 weight for readability

- **Message fade-in** effects

## 🔒 Security Features

### 🛡️ Content Protection
- **Developer tools disabled** in production
- **Right-click context menu blocked**
- **View source protection** for casual users
- **Console logging overridden** to prevent debugging

### 🔐 Data Privacy
- **No data storage** in browser localStorage
- **Session-based** conversation history
- **No tracking** or analytics embedded
- **Secure API communication** with proper headers
