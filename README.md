# AICA - Local AI Chat Interface

A lightning-fast, modern, browser-based chat interface designed to easily communicate with your self-hosted or remote AI language models like Qwen, KIMI, and GLM.

## Architecture Highlights
- **100% Vanilla JS & HTML/CSS**: Zero dependencies, instant loading.
- **Vercel Serverless Proxy**: API routing built-in to prevent CORS issues securely.
- **Dynamic Model Selection**: Instantly swap between LLM configurations.
- **Developer Hardened**: Includes layout protection and console overrides.

*Built for speed, simplicity, and flexibility.*
